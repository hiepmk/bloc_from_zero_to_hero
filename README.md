## Tips and Explanations 👩🏾‍🏫👨🏾‍🏫

### BloC Architecture Suggestion 

```bash
.
├── lib/
│   ├── business_login/
│   │    ├── blocs/
│   │    └── cubits/
│   ├── data/
│   │   ├── data_providers/
│   │   ├── models/
│   │   └── repositories/
│   ├── presentation/
│   │   ├── animations/
│   │   ├── pages/
│   │   └── widgets/
└── main.dart
```
